# SOLT - Science of Life and Transformation

## About This Project

This project was inspired by a passion for making science education accessible and engaging to students. SOLT aims to provide an interactive platform where learners can explore biological concepts through innovative features like specimen rooms, wisdom labs, career mapping, and study materials. By combining visual learning, interactive tools, and structured content, we strive to transform how students engage with science and make learning a more enjoyable and impactful experience.

**Built with:** Google AI Studio

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
